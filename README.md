# Back to Being — voorstel voor NewBe

Een mini-docu-voorstel voor NewBe. Drie founders lopen van Portugal naar Santiago de Compostela en bouwen onderweg, met slimme brillen en hun stem, een app voor mentale gezondheid voor Stichting MIND.

Zelf-bevattend HTML-deck (10 slides) in de Back to Being huisstijl. Open `index.html` of bekijk de live versie via GitHub Pages.

- Fonts: Instrument Serif + Inter (Google Fonts)
- Logo-reveal intro via Lottie (`logo-animation.json`)
- Foto's in `assets/`
